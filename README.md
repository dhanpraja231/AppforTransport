clone the project on android studio.

https://drive.google.com/file/d/1L-DUI8G8Q5aPk5EIeuL4sN7MnIHfsX_4/view?usp=sharing

       set user: "hema"
       set password: "1234"
       
       
       after this  run the application on android studio emulator.
